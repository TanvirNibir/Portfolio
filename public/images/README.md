# Images Folder

Place all your images here.

## Recommended Structure:

- `profile-photo.jpg` or `profile-photo.png` - Your profile picture (for Hero section)
- `projects/` - Project screenshots/images
- `about/` - About section images
- `skills/` - Skills/technology icons
- `backgrounds/` - Background images

## Image Guidelines:

- **Profile Photo**: Square (1:1 ratio), at least 400x400px
- **Project Images**: Recommended 1200x800px or similar
- **Formats**: JPG, PNG, WebP
- **Optimization**: Compress images before uploading for better performance

## Usage in Code:

Reference images like this:
```jsx
<img src="/images/profile-photo.jpg" alt="Description" />
```
